# Cybersecurity Awareness Bot (CYBER BOT)

A Windows Forms application built on .NET 10.0 designed to provide automated, randomized cybersecurity tips regarding email phishing, password safety, and general system security.

1. Open terminal in the project root folder.
2. Run the command: \dotnet run\
