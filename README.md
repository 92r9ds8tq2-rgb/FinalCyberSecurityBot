# Final Cybersecurity Awareness Bot


A Windows Forms (WinForms) desktop chatbot application designed to interactively educate users on cybersecurity best practices.

## 🚀 Features
* **Interactive GUI:** Clean Windows Forms user interface.
* **Voice Greeting:** Audio welcome message upon launching the application.
* **Keyword Recognition:** Parses user input to provide relevant security tips and counter-measures.
* **GitHub CI:** Integrated continuous integration workflow.

## 🛠️ Prerequisites & Installation
To run this project locally, you will need:
* **Visual Studio** (2022 recommended) with the **.NET Desktop Development** workload installed.
* **.NET Framework / .NET Core** (depending on your `.csproj` target).
WinForms chatbot project with GUI, voice greeting, keyword recognition, and GitHub CI.
